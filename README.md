Este proyecto es una aplicación sencilla desarrollada con HTML, CSS y JavaScript, cuyo objetivo es que el usuario intente adivinar un número secreto generado automáticamente por el sistema.

¿En qué consiste el juego?
El usuario debe ingresar un número dentro del rango permitido (por defecto, del 1 al 10) e intentar adivinar cuál es el número secreto generado aleatoriamente.
El sistema le notificará si acertó o si debe seguir intentando. Una vez que el jugador gana, puede iniciar una nueva partida con un solo clic.

Características

Interfaz simple y clara.
Respuestas inmediatas según el intento del usuario.
Generación automática de un nuevo número al reiniciar el juego.
Código organizado y fácil de entender.
